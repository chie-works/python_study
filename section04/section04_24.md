## 24. 辞書型
#### 💻 ターミナルで実行
<div align="right">
  <a href="../README.md#section4">READMEに戻る</a>
</div>


```bash
# 辞書の作成（キーと値のペア）
d = {'x': 10, 'y': 20}
d                       # {'x': 10, 'y': 20}
type(d)                 # <class 'dict'> → 辞書型

# キーを指定して値を取得
d['x']                  # 10
d['y']                  # 20

# 値の更新（キー 'x' の値を変更）
d['x'] = 100
d                       # {'x': 100, 'y': 20}

# 値を文字列に変更することも可能
d['x'] = 'XXXX'
d                       # {'x': 'XXXX', 'y': 20}

# 新しいキーと値を追加
d['z'] = 200
d                       # {'x': 'XXXX', 'y': 20, 'z': 200}

# キーに整数も使える（辞書はキーにイミュータブルな型を使える）
d[1] = 10000
d                       # {'x': 'XXXX', 'y': 20, 'z': 200, 1: 10000}

# dict() コンストラクタで辞書を作成（キーワード引数）
dict(a=10, b=20)        # {'a': 10, 'b': 20}と同じ意味

# dict() コンストラクタで辞書を作成（リストのタプルから）
dict([('a', 10), ('b', 20)])  # {'a': 10, 'b': 20}
```

## 📝 学習のまとめ
- 波括弧 {} を使う
```bash
d = {'x': 10, 'y': 20}
```
- 空の辞書を作成して後から追加
```bash
d = {}
d['x'] = 10
d['y'] = 20
```
- dict() コンストラクタ（キーワード引数）
```bash
d = dict(a=10, b=20)
```
- dict() コンストラクタ（タプルのリストから）
```bash
d = dict([('a', 10), ('b', 20)])
```
- 辞書のコピーや結合
```bash
d1 = {'a': 1}
d2 = {'b': 2}
d = {**d1, **d2}   # 展開して結合
```
