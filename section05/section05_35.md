## 35. 比較演算子と論理演算子
#### 📝 VSCodeで実行
<div align="right">
  <a href="../README.md#section5">READMEに戻る</a>
</div>

```bash
a = 1
b = 2

# 等しいかどうか
print(a == b)

# 等しくないかどうか
print(a != b)

# より小さいかどうか
print(a < b)

# より大きいかどうか
print(a > b)

# 以下かどうか
print(a <= b)

# 以上かどうか
print(a >= b)

# 両方が条件を満たすか
print(a > 0 and b > 0)

# どちらかが条件を満たすか
print(a > 0 or b > 0)
```
